Тема: Веб-додаток для створення власного портфоліо та резюме

Анотація: Сьогодні дуже багато людей шукають роботу в інтернеті, тому з’являються багато ресурсів які дозволяють публікувати професійну інформацію про себе та ділитися своїм досвідом.Але в мережі рідко зустрічаються додатки на яких можна одночасно створити детальне резюме та зручно орагнізувати портфоліо. У своїй роботі я поєднав ці 2 функції тому що, на мою думку професійно складене резюме та коректно складене портфоліо є найважливішими пунктами на які передусім звертають увагу при відборі на вакансію.

Мови програмування: HTML5, CSS3, JavaScript(EcmaScript 12), jQuery(бібліотека JavaScript), PHP, MySQL, JSON

Програмні та апаратні вимоги:
Об'єм жорсткого диска 70 MB
RAM 768 MB
Ядра CPU	1
PHP 7.2

Посилання для перегляду: http://stockexchange.host/public/

Встановлення веб-додатку для роботи на локальному сервері:
1. Завантажити локальний сервер OpenServer стабільна версія 5.3.7
2. Створити в каталозі domains папку під назвою вашого сайту(бажано stockexchange) розташувати папку сайту (Source/)
3. Запустити локальней сервер та перейти на посилання http://127.0.0.1/phpMyAdmin/ створити аккаунт та перейти до авторизації
4. Після авторизації в аккаунті переходимо у вкладку import у якій завантажуємо файл в каталозі(db_dump/)
5. Далі заходимо у вкладку (User accounts) та створюємо користувача
6. Далі заходимо в файл (init.php) за наступним шляхом (назва вашого сайту/app/init.php)
7. У цьому файлі знаходимо пункт mysql та змінюємо там наступні параметри за назвами зміних списка
8. Перезавантажуємо сервер і користуємося
